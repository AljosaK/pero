# DataMerger

### Codes you can run
```
npm run prepare // nvm use 9.4.0
```
```
npm run build // browserify index.js > bundle.js
```
```
npm run watch // watchify index.js -o bundle.js
```
```
npm run serve // npm run watch & ws
```
